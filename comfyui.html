Гид по ComfyUI

ComfyUI — мощный инструмент для генерации изображений на основе Stable Diffusion. В отличие от обычных интерфейсов, здесь ты сам соединяешь блоки-ноды в цепочку и управляешь каждым шагом генерации.

🧩

⚙️ Установка

ComfyUI работает локально на твоём компьютере — данные никуда не отправляются. Для работы нужна видеокарта NVIDIA (от GTX 1060 6GB) или AMD, или процессор (медленнее).

1

Скачай ComfyUI

Перейди на страницу релизов ComfyUI на GitHub и скачай архив для Windows.

https://github.com/comfyanonymous/ComfyUI/releases

Скачивай файл: ComfyUI_windows_portable_nvidia.7z (для NVIDIA) или ComfyUI_windows_portable_cpu.7z (только процессор)

2

Распакуй архив

Распакуй .7z архив в любую папку. Понадобится 7-Zip если не установлен.

💡 Избегай пути с кириллицей или пробелами. Лучше всего: C:\ComfyUI или D:\AI\ComfyUI
3

Скачай модель

Без модели ComfyUI не генерирует. Скачай любую модель в формате .safetensors с Hugging Face или Civitai и положи в папку:

ComfyUI\models\checkpoints\

Для начала подойдёт dreamshaper_8.safetensors (универсальный) или v1-5-pruned-emaonly.safetensors (оригинальный SD 1.5)

4

Запусти ComfyUI

Дважды кликни на файл запуска в корне папки:

run_nvidia_gpu.bat — для видеокарт NVIDIA run_cpu.bat — если нет GPU или AMD

Откроется чёрное окно терминала. Подожди — когда появится строка To see the GUI go to: http://127.0.0.1:8188 — открой этот адрес в браузере.

💡 Не закрывай окно терминала — ComfyUI работает через него
5

Готово!

В браузере откроется интерфейс ComfyUI с базовым воркфлоу. Нажми Queue Prompt — и первая генерация запустится.

1

Установи Homebrew и Python

Открой Terminal и выполни:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install python@3.10
2

Скачай ComfyUI

git clone https://github.com/comfyanonymous/ComfyUI cd ComfyUI
3

Установи зависимости

pip install -r requirements.txt

Для Mac с чипом Apple (M1/M2/M3) добавь поддержку Metal:

pip install torch torchvision torchaudio
4

Положи модель и запусти

Скопируй модель (.safetensors) в папку models/checkpoints/, затем:

python main.py --force-fp16

Открой в браузере: http://127.0.0.1:8188

💡 Флаг --force-fp16 нужен для экономии памяти на Mac
1

Установи Python 3.10+

Скачай с python.org и установи. Убедись что Python добавлен в PATH.

python --version # должно показать 3.10+
2

Клонируй репозиторий

git clone https://github.com/comfyanonymous/ComfyUI cd ComfyUI
3

Установи PyTorch и зависимости

Для NVIDIA (CUDA 11.8):

pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118

Затем общие зависимости:

pip install -r requirements.txt
4

Запусти

python main.py
💡 Полезные флаги: --lowvram (для GPU меньше 4GB), --cpu (без GPU), --port 8080 (другой порт)

💡 Базовые концепции

Прежде чем лезть в ноды — несколько важных идей, без которых будет сложно разобраться.

🔲

Нода (блок)

Каждый прямоугольник на холсте — это нода. Одна нода выполняет одну задачу: загружает модель, генерирует изображение, сохраняет результат. Ноды соединяются между собой «проводами».

🔌

Входы и выходы

У каждой ноды есть слоты: слева — входы (что нода получает), справа — выходы (что нода отдаёт дальше). Соединяй выход одной ноды со входом другой — они должны быть одного типа (цвет совпадает).

🎨

Типы данных (цвета)

Каждый тип данных окрашен в свой цвет: фиолетовый — MODEL, голубой — CLIP, розовый — VAE, зелёный — LATENT, оранжевый — IMAGE. Нельзя подключить LATENT куда ожидается IMAGE.

🌫️

Латентное пространство

Нейросеть работает не с пикселями напрямую, а в «сжатом» математическом пространстве — латентном. VAE Encode переводит изображение туда, KSampler генерирует там, VAE Decode возвращает обратно в пиксели.

📝

Позитивный и негативный промпт

Нужны две ноды CLIP Text Encode: одна описывает что хочешь увидеть, другая — чего не хочешь. Негативный промпт помогает убирать типичные проблемы: размытость, лишние конечности, плохое качество.

🎲

Seed и воспроизводимость

Seed — число, задающее «случайность» генерации. Одинаковый seed + одинаковые настройки = одинаковое изображение. Это позволяет экспериментировать, меняя только один параметр за раз.

⚡

Queue Prompt

Кнопка запуска генерации. ComfyUI выполняет граф нод слева направо — сначала загрузчики, потом генерация, в конце сохранение. Можно поставить несколько заданий в очередь.

📂

Структура папок

models/checkpoints/ — основные модели
models/loras/ — LoRA файлы
models/vae/ — VAE модели
models/controlnet/ — ControlNet модели
output/ — готовые изображения

🗂️ Готовые воркфлоу

Типовые сборки нод под разные задачи. Изучи какие ноды нужны для каждого сценария — это поможет собрать собственный воркфлоу.

Начинающий

🖼️ Text to Image

Базовый воркфлоу: генерация изображения из текстового описания с нуля. Именно с него начинают все.

Load Checkpoint
CLIP Text Encode × 2 (+ и −)
Empty Latent Image
KSampler
VAE Decode
Save Image
Начинающий

🔄 Image to Image

Берёт существующее изображение и изменяет его по промпту. Сила изменения регулируется параметром denoise в KSampler.

Load Checkpoint
Load Image
VAE Encode
CLIP Text Encode × 2
KSampler (denoise: 0.5–0.8)
VAE Decode → Save Image
Средний

🔭 Hires Fix

Двухэтапная генерация: сначала в маленьком разрешении, потом апскейл и доработка деталей. Позволяет получить большие изображения без артефактов.

Load Checkpoint + промпты
Empty Latent Image (512×512)
KSampler (шаг 1)
Latent Upscale (до 1024×1024)
KSampler (шаг 2, denoise: 0.4)
VAE Decode → Save Image
Средний

🔬 Upscale нейросетью

Увеличивает готовое изображение в 4× через специализированную upscale-модель (RealESRGAN, 4x-UltraSharp).

Load Image
Load Upscale Model
Image Upscale with Model
Save Image
Средний

🎨 LoRA стилизация

Добавляет в воркфлоу LoRA — модуль для конкретного стиля или персонажа. Можно стекировать несколько LoRA с разной силой.

Load Checkpoint
Load LoRA (сила: 0.7–1.0)
CLIP Text Encode × 2
Empty Latent Image
KSampler → VAE Decode → Save
Продвинутый

🕹️ ControlNet по контурам

Генерирует изображение с точным соблюдением контуров референса. Используй Canny ControlNet — он извлекает края из фото.

Load Checkpoint + Load ControlNet
Load Image (референс)
Apply ControlNet (canny)
Conditioning Combine
KSampler → VAE Decode → Save
Продвинутый

🖌️ Inpainting

Закрашивает выбранную область изображения по промпту. Нужно нарисовать маску поверх той части, которую хочешь изменить.

Load Image + маска
VAE Encode (for Inpaint)
CLIP Text Encode × 2
KSampler (denoise: 1.0)
VAE Decode → Save Image
Продвинутый

🌅 Outpainting

Дорисовывает то, что находится за краями изображения. Позволяет расширить фотографию или арт в любую сторону.

Load Image
Image Pad for Outpaint
VAE Encode + маска
KSampler (denoise: 1.0)
VAE Decode → Save Image

🔌 Кастомные ноды

Помимо стандартных нод, сообщество создало тысячи расширений. Устанавливаются через ComfyUI Manager — его ставят первым делом.

🗂️

ComfyUI Manager

ltdrdata

Менеджер расширений для ComfyUI. Позволяет устанавливать, обновлять и удалять кастомные ноды прямо из интерфейса — без командной строки. Устанавливай первым.

🕹️

ControlNet Preprocessors

Fannovel16

Набор нод для подготовки референсов ControlNet: извлечение контуров (Canny, Lineart), определение поз (OpenPose), карты глубины (MiDaS, Zoe) и многое другое.

🖼️

IPAdapter Plus

cubiq

Перенос стиля и внешности с одного изображения на другое. Скажи модели «сделай как на этой картинке» — она перенесёт стиль, освещение или лицо персонажа.

🎬

AnimateDiff

Kosinkadink

Создаёт анимации и короткие видео из Stable Diffusion. Добавляет «движение» к генерации — получаются плавные зацикленные анимации.

✂️

WAS Node Suite

WASasquatch

Огромный набор утилит: работа с масками, текстом, изображениями, цветами, файлами. Более 100 дополнительных нод для тонкой настройки воркфлоу.

🔤

ComfyUI Impact Pack

ltdrdata

Детектирование лиц, автоматическое inpainting проблемных зон, сегментация объектов. Помогает исправлять дефекты рук и лиц автоматически.

🧩

SDXL Prompt Styler

twri

Библиотека стилей для промптов. Выбираешь стиль из списка (кинематографический, акварель, 3D-рендер и т.д.) — нода автоматически добавляет нужные теги к промпту.

🌐

ComfyUI-VideoHelperSuite

Kosinkadink

Загрузка, обработка и сохранение видео. Нужен для AnimateDiff и других видео-воркфлоу: разбивает видео на кадры и собирает обратно.

💡 Все кастомные ноды устанавливаются через ComfyUI Manager → Install Custom Nodes → поиск по названию → Install. После установки нужно перезапустить ComfyUI.
