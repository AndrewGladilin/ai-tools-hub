# 🤖 AI News Digest Agent

Ежедневный агент для сборки и суммирования новостей по ИИ/ML с отправкой в Telegram канал.

## Особенности

- 🌐 **9+ источников новостей**: Habr, HackerNews, ArXiv, Anthropic, OpenAI, Medium, TechCrunch, Product Hunt, The Verge, Lobsters
- 🧠 **Groq LLM для суммирования**: Быстрое и бесплатное суммирование каждой новости
- 🇷🇺 **Полностью на русском**: Перевод заголовков и суммаризация на русском языке
- 📱 **Отправка в Telegram**: Красивый дайджест прямо в твой канал
- ⚙️ **Полностью автоматизировано**: Cron job для ежедневного запуска
- 💰 **Полностью бесплатно**: Groq API + твой VPS

## Требования

- Python 3.8+
- pip
- Git

## Установка на VPS

### 1. Клонируем репо и переходим в папку

```bash
cd ~/ai-tools-hub/news-agent
```

### 2. Создаём виртуальное окружение

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Устанавливаем зависимости

```bash
pip install -r requirements.txt
```

### 4. Копируем .env.example в .env и заполняем

```bash
cp .env.example .env
nano .env
```

Заполни значения (получи ключи с сайтов):
```
GROQ_API_KEY=<твой_groq_ключ>
TELEGRAM_BOT_TOKEN=<твой_telegram_токен>
TELEGRAM_CHANNEL=neuro_andrew
```

### 5. Тестируем скрипт

```bash
python3 main.py
```

Должна увидеть логи и сообщение в телеграм-канале.

## Настройка Cron для ежедневного запуска

### 1. Открываем crontab

```bash
crontab -e
```

### 2. Добавляем строку для ежедневного запуска в 9:00 UTC

```bash
0 9 * * * cd /path/to/news-agent && source venv/bin/activate && python3 main.py >> news_agent_cron.log 2>&1
```

Замени `/path/to/news-agent` на полный путь до папки (например, `/home/username/ai-tools-hub/news-agent`)

### 3. Проверяем установку

```bash
crontab -l
```

Должна видеть твою строку в списке.

## Как это работает

1. **Парсинг** (`sources.py`): Скрипт достаёт свежие новости с 9 источников через RSS
2. **Дедупликация**: Удаляет дубликаты по названию и домену
3. **Суммирование** (`summarizer.py`): Отправляет каждую новость в Groq LLM для однострочной суммаризации
4. **Форматирование**: Красиво оформляет дайджест с ссылками и эмодзи
5. **Отправка** (`telegram.py`): Постит в твой телеграм-канал

## Логирование

Логи пишутся в `news_agent.log` и на консоль. Очень полезно для отладки:

```bash
tail -f news_agent.log
```

## Решение проблем

### "GROQ_API_KEY not found"
- Проверь, что скопировал `.env.example` в `.env`
- Проверь, что заполнил `GROQ_API_KEY` в `.env`

### "Telegram API error"
- Проверь, что токен правильный
- Проверь, что бот добавлен в канал с правами на отправку сообщений

### "No news items found"
- Это нормально, значит ни один источник не вернул новостей в этот момент
- Проверь интернет-соединение

## Кастомизация

### Изменить время запуска
В `crontab -e` измени первую часть:
```bash
# Каждый день в 10:00 UTC
0 10 * * * ...

# Каждый день в 6:00 UTC
0 6 * * * ...

# Только в будни в 9:00 UTC
0 9 * * 1-5 ...
```

### Добавить свой источник
1. Добавь метод в класс `NewsFetcher` в `sources.py`
2. Вызови его в методе `fetch_all()`

### Изменить количество новостей
В `main.py` измени:
```python
top_news = fetcher.get_top_news(count=15)  # Вместо 10
```
